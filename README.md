Argument-based Consistency (ArC) in Toxicity Explanations of LLMs
=================================================================

This repository contains supplementary details for the Argument-based Consistency Poster presented at the DSI Talent Showcase Event on January 22, 2026.

.. image:: https://github.com/raam93/ArC_at_DSI/blob/main/arc_overall_flow.png 
  :align: center
  :width: 400px

Note: This paper will be published at EACL 2026. An earlier version is available on arXiv (https://arxiv.org/pdf/2506.19113), where the framework was originally called "Human-Aligned Faithfulness (HAF)" before being renamed to "Argument-based Consistency" during peer review. Code for all experiments is available at https://github.com/uofthcdslab/HAF and will be updated by January 2026.
