Argument-based Consistency (ArC) in Toxicity Explanations of LLMs
=================================================================

This repository contains supplementary details for the Argument-based Consistency Poster presented at EACL 2026 and at the DSI Talent Showcase Event.

.. image:: https://github.com/raam93/ArC_at_DSI/blob/main/arc_overall_flow.png 
  :align: center
  :width: 400px

**Note:** An earlier version of this work, available on arXiv (https://arxiv.org/pdf/2506.19113), was originally called "Human-Aligned Faithfulness (HAF)" before being renamed to "Argument-based Consistency" during peer review. Code for all experiments is available at https://github.com/uofthcdslab/ArC.
